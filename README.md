# msbtlib

🚧 Work in progress.

Library for reading, writing, and converting MSBT files.

## Status
- [x] Initial structure
- [x] Basic parser
- [x] Basic file writing support
- [X] Convert MSBT to dict
- [ ] Convert MSBT to JSON
- [ ] Documentation

## Roadmap
- Improve parser robustness
- Increase test coverage
- Add CLI support

## Tests

The tests require MSBT files. You need to extract them yourself from your game copy:

1. Place the MSBT file in `tests/data/`.
2. Name it `example.msbt`.
